# Product

## Register

product

## Users

นักพัฒนาและผู้ใช้ Codex CLI ที่มีหลายบัญชี ต้องการเห็นโควตาปัจจุบันและเปิดบัญชีที่เหมาะสมโดยไม่จัดการไฟล์รับรองสิทธิ์ด้วยตนเอง

## Product Purpose

Codex Switch รวมการติดตามโควตา การแยกโปรไฟล์ และการเปิด Codex ไว้ในแอปเดสก์ท็อปขนาดเล็ก ความสำเร็จคือผู้ใช้เปิดแอปแล้วตัดสินใจได้ภายในไม่กี่วินาทีว่าควรใช้บัญชีใด

## Brand Personality

สุขุม ชัดเจน รวดเร็ว ใช้ภาษาตรงไปตรงมาและไม่ตกแต่งเกินหน้าที่

## Anti-references

ไม่เป็นแดชบอร์ด SaaS ที่มีการ์ดจำนวนมาก ไม่ใช้ glassmorphism, gradient, animation เพื่อการตกแต่ง หรือหน้าตั้งค่าที่ซับซ้อน

## Design Principles

- แสดงข้อมูลที่ใช้ตัดสินใจก่อนข้อมูลประกอบ
- ทุกสถานะต้องบอกแหล่งข้อมูลและความสดของข้อมูล
- งานประจำควรเสร็จได้ในหนึ่งหรือสองคลิก
- ทำงานเบื้องหลังเท่าที่จำเป็นและหยุดเมื่อไม่มีประโยชน์
- ไม่ซ่อนความเสี่ยงของ private API จากผู้ใช้

## Accessibility & Inclusion

เป้าหมาย WCAG 2.2 AA รองรับคีย์บอร์ด contrast อย่างน้อย 4.5:1 และ reduced motion สีไม่ใช่ตัวบอกสถานะเพียงอย่างเดียว
